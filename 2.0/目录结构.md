## vue 目录结构
```shell
vue 2.6.14
├─dist                # 打包后文件所在的目录
├─examples            # demo
├─flow                # vue2.0使用flow进行静态类型检查，类型检测相关的目录
├─packages            # 工程包
├─scripts             # 打包相关的配置文件
├─src                 # 源码目录
│  ├─compiler         # 编译相关
│  │  ├─codegen         # 根据ast(抽象语法树)生成render函数
│  │  ├─directives      # 生成render函数所需的指令
│  │  └─parser          # 模板解析 
│  ├─core             # 核心代码
│  │  ├─components      # 全局组件
│  │  ├─global-api      # 全局api
│  │  ├─instance        # 实列相关内容，包括实列方法，生命周期
│  │  ├─observer        # 双向数据绑定相关的源码
│  │  ├─util            # 工具方法
│  │  └─vdom            # 虚拟dom相关
│  ├─platforms        # 跨平台支持
│  │  ├─web             # web端独有的文件
│  │  │  ├─compiler     # 编译阶段需要处理的指令和模块
│  │  │  ├─runtime      # 运行阶段需要处理的组件、指令、模块
│  │  │  ├─server       # 服务器渲染相关
│  │  │  └─util         # 工具文件
│  │  └─weex            # weex端文件
│  ├─server           # 服务端渲染
│  ├─sfc              # vue文件解析
│  └─shared           # 共享工具相关
├─test              # 测试用例
```

`vue` 主要源码在src目录下
